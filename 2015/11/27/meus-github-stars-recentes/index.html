Meus GitHub Stars mais recentes

Published on Friday, November 27th 2015 at 15:50
Last updated on Thursday, November 14th 2019 at 12:52

Como algumas pessoas recentemente me perguntaram como consigo saber sobre novos frameworks, recursos ou mesmo boas práticas de desenvolvimento com javascript resolvi colocar nesse post alguns repositórios e pessoas que andei seguindo no github nos últimos meses e que acabam me ajudando nesse processo de manter-se atualizado principalmente em relação a javascript e .NET

Alguns Follows

Eric Elliot - https://github.com/ericelliott

Se tem um cara que eu quero ser quando crescer é esse aí, exceto pelo visual serial killer.

Acompanho ele pelo twitter também e sempre aparecem posts muito bons em seu blog, posts enormes diga-se de passagem mas sempre muito interessantes.

O Eric mantem alguns bons repositórios como o https://github.com/ericelliott/essential-javascript-links contendo uma enorme quantidade de boas referências sobre JS, a partir desse repositório conheci muitos outros projetos e pessoas.

Rick Waldron - https://github.com/rwaldron

Commiter de excelentes projetos em Node como o http://johnny-five.io/, participante ativo do core do JQuery, possui excelente apresentações no YouTube e bons posts espalhados pela web

TJ Holowaychuk - https://github.com/tj

Uma quantidade grande de repositórios com excelentes projetos, o cara é fera.

Vincent Orr - https://github.com/Cmdv

Quando comecei a seguir o Vincent percebi que ele sempre dá Star em excelentes projetos no github e isso coloca na minha home page do GitHub muita novidade.

Sindre Sorhus - https://github.com/sindresorhus

Um dos criadores do Yeoman e da famosa TodoMVC.

Possui uma excelente conta no twitter - https://twitter.com/sindresorhus, sempre com muita coisa útil sendo publicada.

.NET repositories

  • Awesome .NET! - A collection of awesome .NET libraries, tools, frameworks, and software.

JavaScript repositories

  • Awesome - mega lista com excelente links para frameworks, linguagens, arquitetura de software e projetos
  • Idiomatic.JS - JavaScript de forma Consistente e Idiomática
  • Airbnb JavaScript Style Guide
  • Essencial JavaScript Links
  • GitHub Cheat Sheet - excelente lista com dicas e recursos de git e do GitHub
  • github-email - Obtém o email da conta de um perfil no github mesmo não sendo público

GitHub

Após seguir boas referências no GitHub sempre dê uma olhada na home page dele (versão autenticada), a listagem de atividades sempre vai mostrar projetos que essa galera acaba seguindo ou recomendando, logo, para mim sempre é uma boa referência dar uma olhada e ver o que tem de novidade por lá.

É isso, até mais.
